# 7 Hábitos - Site (React + Vite + Tailwind)

Este projeto é a versão React do site de vendas do curso **7 Hábitos que Transformam Sua Vida em 30 Dias**.
O botão de compra já aponta para: https://pay.kiwify.com.br/d2DDOct

## Como usar

1. Instale dependências:
```bash
npm install
```

2. Rodar em desenvolvimento:
```bash
npm run dev
```

3. Build para produção:
```bash
npm run build
npm run preview
```

> Obs: se preferir, você pode publicar este projeto diretamente no **Vercel** conectando com o seu repositório GitHub — Vercel detecta o Vite automaticamente.
